# Orders

## Project setup
```
yarn install
```

### Start backend

1. `bin/backend-down.sh`
2. `bin/backend-up.sh`
3. `bin/backend-data.sh` *(Optional to add starting data)*

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
